xmodule2-examples-security-oauth2-advance-clients: 基于Spring-Security的OAuth2客户端示例
========================================================


Quickstart
-----------

1.安装并启动Keyclock8.0.0
2.启动xmodule2-examples-security-oauth2-advance-clients
3.执行http://localhost:18081/api/authserver/reinit来初始化keycloak配置
4.启动xmodule2-examples-security-oauth2-advance-resource-server
5.访问http://localhost:18081/index来测试基于Client Credentials、Authorization Code、Resource Owner Password Credentials等模式下的OAuth2客户端实例
